# prisma-basic

# Project Setup
```
npm install
```

# Project Run
```
npm start
```

# How to init prisma for the first time
```
npx prisma init
```
It will create `/prisma/schema.prisma` and `.env`. Change those two files as needed.

# Pull from existing database
```
npx prisma db pull
```

# generate prisma client
```
npx prisma generate
```

# generate migrate
```
npx prisma migrate
```
